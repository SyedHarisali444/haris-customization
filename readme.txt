A clean starter plugin to practice WordPress PHP customizations: hooks, CPT, meta, REST, AJAX, and WooCommerce examples.

== Installation ==
1. Upload the ZIP via Plugins → Add New → Upload Plugin.
2. Activate the plugin.
3. Find the “Haris Toolkit” page in the admin menu.

== Features ==
* Hooks: actions + filters baseline.
* CPT + taxonomy with meta box and admin column.
* Shortcode `[haris_badge text="Pro"]`.
* REST route `GET /wp-json/haris/v1/ping`.
* AJAX scaffold (localized nonce + ajaxUrl).
* WooCommerce tweaks (optional; loads only if Woo is active).
